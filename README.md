## JavaScript Fundamentals

JavaScript is the foundation of the entire ecosystem covered in this project. It is a dynamic and essential language for modern development.

### Key Features:
- Dynamic typing  
- Runs on both browser and server (Node.js)  
- Asynchronous programming (Promises and async/await)  

### Example:

```javascript
class Point {
  constructor(x, y) {
    this.x = x;
    this.y = y;
  }

  r() {
    return Math.sqrt(this.x * this.x + this.y * this.y);
  }
}

const p = new Point(1, 1);
console.log(p.r());
```
