# JavaScript, React e React Native — Estudo Guiado com NotebookLM

Este repositório apresenta um estudo estruturado sobre JavaScript, React e React Native, utilizando a ferramenta NotebookLM como apoio para organização, análise e geração de conteúdo.

---

## Contexto e Objetivos

O tema escolhido para este estudo foi o ecossistema JavaScript, com foco no desenvolvimento web e mobile utilizando React e React Native.

Os principais objetivos deste projeto são:

- Consolidar os fundamentos da linguagem JavaScript  
- Compreender a construção de interfaces com React  
- Explorar o desenvolvimento mobile com React Native  
- Entender as evoluções modernas de arquitetura e performance  
- Utilizar inteligência artificial como ferramenta de apoio ao aprendizado  
- Desenvolver um material estruturado para consultas futuras

Durante o desenvolvimento do estudo, foram utilizados diversos prompts no NotebookLM para extrair, organizar e aprofundar o conhecimento.

### Prompts utilizados

- "Explique os fundamentos do JavaScript de forma clara e estruturada"

- "O que é o Hermes Engine e quais suas vantagens?"

### Estratégia aplicada

- Uso de prompts específicos e direcionados  
- Refinamento progressivo das perguntas  
- Separação por temas (fundamentos, arquitetura, aplicações)

- Respostas genéricas ao utilizar prompts vagos  
- Necessidade de reformular perguntas para maior precisão  
- Organização das informações geradas pela IA  
- Identificação de conteúdo redundante  

### Aprendizados

- A qualidade do prompt impacta diretamente na qualidade da resposta  
- Contexto é essencial para obter respostas mais técnicas  
- A IA funciona melhor como apoio, não como substituição do estudo

## Miniguia de Estudo

### JavaScript

JavaScript é uma linguagem essencial para o desenvolvimento moderno, utilizada tanto no frontend quanto no backend.

Principais conceitos:

- Tipagem dinâmica  
- Programação assíncrona (Promises e async/await)  
- Manipulação de objetos e arrays  

---

### React

React é uma biblioteca JavaScript utilizada para construção de interfaces web modernas.

Principais conceitos:

- Componentização  
- State e Props  
- Virtual DOM  

Aplicações:

- Dashboards  
- Sistemas web  
- Single Page Applications  

---

### React Native

React Native permite o desenvolvimento de aplicações mobile multiplataforma.

Principais vantagens:

- Código compartilhado entre Android e iOS  
- Alta produtividade  
- Integração com recursos nativos  

---

### Arquitetura Moderna

O ecossistema evoluiu significativamente, trazendo melhorias importantes:

- Hermes Engine (melhor performance)  
- Nova Arquitetura (substituição da bridge)  
- TurboModules e Fabric  
- Nitro Modules (alto desempenho e melhor uso de memória)  

---

## Glossário

- **JavaScript**: Linguagem de programação utilizada no desenvolvimento web e mobile  
- **React**: Biblioteca para construção de interfaces web  
- **React Native**: Framework para desenvolvimento mobile  
- **Hermes**: Motor JavaScript otimizado para dispositivos móveis  
- **TurboModules**: Sistema de comunicação eficiente entre JavaScript e código nativo  
- **Fabric**: Sistema moderno de renderização do React Native  
- **Nitro Modules**: Evolução dos módulos nativos com maior performance
